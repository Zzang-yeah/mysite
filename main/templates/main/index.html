데이터 설명

캘리포니아 인구조사 데이터를 분석하여 주택 가격에 가장 영향을 많이 미치는 요소 분석
특성

  • longitude : 경도

  • latitude : 위도
  • housing_median_age : 주택 평균 연식
  • total_rooms : 방의 갯수
  • total_bedrooms : 침실의 갯수
  • popliation : 인구
  • households : 가구
  • median_income : 중간 소득
  • median_house_value : 주택 평균 가치
  • ocean_proximity : 바다에 근접한 정도

위도&경도와 주택 가격의 관계

빨간색은 높은 가격, 파란색은 낮은 가격, 큰 원은 인구가 밀집된 지역
주택 가격은 지역과 인구 밀도에 관련이 매우 크다는 것을 알 수 있음
해안선근처가 가격이 비싼 것을 알 수 있음

중간 소득, 방 갯수, 주택 평균 연식과 주택 가격의 관계

대각선 방향은 자기 자신에 대한 것이라서 볼 필요X
중간 소득과 주택가격이 퍼지지 않은 오른쪽 위를 향하는 대각선 방향을 나타내는 것으로 보아 강한 양의 상관관계를 가지는 것을 알 수 있음
많이 퍼진 모양을 가지고 있으면 선형적인 관계가 없음
